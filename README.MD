# My little community



## Install

```bash
git clone https://github.com/basaxter/My_little_community
cd My_little_community
yarn install # (or npm install)
```

## Start in dev:
```bash
yarn dev # (or npm run build && npm run start)
```

## Start in production:
```bash
yarn build && yarn start #(or npm run build && npm start)
```
